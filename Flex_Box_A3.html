FlexBox

Objective.

This website is made to visualize the use of the following:
  • Use of Flex box
  • Use of Grid
  • Difference between flexbox and Grid
  • What settings are needed and how they affect the layout
  • How the layout responds to changes in screen size
  • List the different features and how to use them

    • Answers

    The answer to the above questions:
  • Flexbox is mainly used for 1-D layouts. If you want to put the datas in rows or columms. It is especially great with buttons.
  • Grid is mainly used for 2-D layouts (rows and columns). It is great for full page layouts, dashboards or galleries in a web page.
  • Mainly 4 settings are needed. They are: "flex-wrap", "justify-content", "align-items" and "gap". These settings are one of the mnay that help in resize and adapt the content when the size changes.

    • Features

    Here will be the different features.
  • flex-grow, flex-shrink and flex-basis: grow checks how much it can grow, shrinks controls how much the item should shrink when space is tight and basis sets initial size before everything.
  • order: Change item order.
  • align-self: overrides alignment per item.
